# myInvesting

### 주식 관련 유의미한 뉴스들을 수집해서 보여주는 개인 투자 보조 사이트.

## 주요 기능 (Features)

1. 메인페이지에서 주식티커를 통해 각 주식에 대하여 정보를 찾아볼 수 있다.
2. 각 종목 별 페이지에서 관심종목을 지정한다.
3. 메인페이지에서 관심종목 바로가기, 관심종목 뉴스들을 한 눈에 볼 수 있다.
4. 메인페이지에서 뉴스를 클릭하면 자세한 뉴스 내용을 볼 수 있다.
